# Pokémon Battle Compass

## v0.1.0-alpha.1

Initial desktop Alpha release

### Features
- Battle Compass
- My Team editor
- Journey onboarding
- Reference dialogs
- Matchup Strength meter
- Ability-aware recommendations
- Full Analysis
- Persistent local Journey storage

### Known limitations
- Desktop only
- Singles battles only
- Pokémon Sword only
- No import/export yet
